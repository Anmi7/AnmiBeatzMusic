# Assets Directory Structure

This directory contains all the media assets for the AnmiBeatzMusic website (anmibeatz.com).

## Directory Structure

```
assets/
└── images/
    ├── backgrounds/     # Background images for different sections
    │   ├── hero-background.jpg
    │   └── about-background.jpg
    │
    ├── logos/          # Logo files
    │   ├── anmi-beatz-logo.png
    │   └── anmi-beatz-logo-white.png
    │
    └── music-covers/   # Album and beat cover artwork
        ├── beat1-cover.jpg
        ├── beat2-cover.jpg
        └── album-cover.jpg
```

## Image Guidelines

### Backgrounds
- Place all background images in the `backgrounds` directory
- Use descriptive names like `hero-background.jpg`, `about-background.jpg`
- Recommended size for hero background: 1920x1080px
- Format: JPG or PNG

### Logos
- Place all logo files in the `logos` directory
- Include both light and dark versions if needed
- Use transparent PNG format
- Recommended size: 200x200px minimum

### Music Covers
- Place all music cover artwork in the `music-covers` directory
- Name files according to the beat/album name
- Recommended size: 1000x1000px
- Format: JPG or PNG

## Naming Convention
- Use lowercase letters
- Use hyphens (-) to separate words
- Include the purpose in the filename
- Example: `beat-name-cover.jpg`, `section-name-background.jpg`

## File Formats
- Use JPG for photographs and complex images
- Use PNG for logos and images that require transparency
- Optimize all images for web use to ensure fast loading times
